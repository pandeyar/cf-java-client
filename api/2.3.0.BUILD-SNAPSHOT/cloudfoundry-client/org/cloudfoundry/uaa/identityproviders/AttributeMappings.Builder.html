Class AttributeMappings.Builder

  • java.lang.Object
    • org.cloudfoundry.uaa.identityproviders.AttributeMappings.Builder
  • Enclosing class:
    AttributeMappings

    public static final class AttributeMappings.Builder
extends Object
    Builds instances of type AttributeMappings. Initialize attributes and then invoke the build() method to create an immutable instance.

    Builder is not thread-safe and generally should not be stored in a field or collection, but instead used immediately to create instances.
