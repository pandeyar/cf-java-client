Class TokenPolicy.Builder

  • java.lang.Object
    • org.cloudfoundry.uaa.identityzones.TokenPolicy.Builder
  • Enclosing class:
    TokenPolicy

    public static final class TokenPolicy.Builder
extends Object
    Builds instances of type TokenPolicy. Initialize attributes and then invoke the build() method to create an immutable instance.

    Builder is not thread-safe and generally should not be stored in a field or collection, but instead used immediately to create instances.
